# Rock, Paper, Scissors Game

- ### Interactive Rock, Paper, Scissors game built in Python

- ### Player competes against the computer







# Features

- ### Input validation: Only allows Rock, Paper, or Scissors (case-insensitive)

- ### Continuous gameplay until the player chooses to quit

- ### Randomized computer choice each round




# Example Output

```
Select Rock/Paper/Scissors [Q to quit]: Rock
Player: Rock  CPU: Scissors
Player Wins!

Select Rock/Paper/Scissors [Q to quit]: Paper
Player: Paper  CPU: Rock
Player Wins!

Select Rock/Paper/Scissors [Q to quit]: Q
Thanks for playing!

```
